Dialog
- เป็น layer ขึ้นมาบนหน้าเว็บ
- Modal
    |- บังคับให้ผู้ใช้ตอบก่อน ไม่สามารถทำอย่างอื่นได้
    |- ใช้บังคับ user ให้ตอบก่อน
- Non-Modal
    |- ไม่บังคับ user ว่าต้องตอบ
    |- สามารถทำอย่างอื่นได้ ในขนาดที่ dialog ยังโชว์อยู่


ความรู้ที่ได้รับ
- ทำความเข้าใจเกี่ยวกับ dialog
- นำไปใช้กับงานวิชา integrated 
- สามารถเลือกใช้ ประเถท dialog ตามความเหมาะสม


ทบทวนเนื้อหา
- ทำความเข้าใจมากขึ้น
- ฝึกปฎิบัติมากขึ้นเช็คความเข้าใจ
